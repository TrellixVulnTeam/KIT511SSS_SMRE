511 Student Service System
====



Version
----
Node.js - 12.18.2

You can use version from 12 - 14
Do not use version 15. 
Because it is too new, 
it will lead to sass errors.

To run the project,

1. npm install
2. npm run dev


If you can not run the program, 
try this way:

cd到项目目录
然后依次执行下面的命令

rm -rf node_modules
rm package-lock.json
npm cache clear --force
npm install
